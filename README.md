## 🧠 Business Problem

Food delivery platforms generate massive data — but without analysis, it's useless.

This project answers:

* Which restaurants drive the most revenue?
* What cuisines are trending?
* Are customers satisfied?
* Where are delivery delays happening?
* Which payment methods dominate?

---

## 🚀 Solution

An **interactive Power BI dashboard** that:

* Tracks KPIs in real-time
* Visualizes trends & patterns
* Enables filtering for deep analysis
* Supports data-driven decisions
